File: `algorithmia.json`

```
{
  "apiKey": "hashkey_here"
}
```

```
{
  "apikey": "hash_apikey",
  "iam_apikey_description": "description of apikey",
  "iam_apikey_name": "name of apikey",
  "iam_role_crn": "...",
  "iam_serviceid_crn": "...",
  "url": "..."
}
```
